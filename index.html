Distribution of chromosomes and extrachromosomal replicons (ERs)

This supplementary dataset contains interactive scatterplots showing the distribution of chromosomes and ERs across different bacterial taxa (phyla, classes, families). Each plot presents GC content (x-axis) versus replicon size (y-axis), color-coded by replicon type. Two versions are available for each group: one using a linear y-axis and another using a logarithmic scale.
